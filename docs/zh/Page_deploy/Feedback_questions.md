# 反馈问题和提供建议



## 反馈问题

请在下述联系方式里选择一种（推荐使用Github 在issues里提出反馈，并选择bug标签），写上你遇到的问题和重现方式，如果问题被修复，将被加入贡献名单

## 提供建议

请在下述联系方式里选择一种，（推荐使用Github 在issues里提出建议，并在标题标开头上 " proposal: "），写出你提供的建议和建议实现方式，如果建议被采纳，将被加入贡献名单
